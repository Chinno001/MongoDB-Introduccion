# MongoDB-Introduccion
Ejercicios de búsquedas avanzadas en MongoDB. Integración de MongoDB con Python mediante PyMongo.

En esta unidad, explorarás cómo seleccionar información en MongoDB usando filtros, 
condiciones y documentos anidados. Aprenderás a usar patrones de expresiones regulares 
para búsquedas avanzadas en campos específicos. 
También conocerás cómo integrar MongoDB con Python mediante la librería PyMongo, 
permitiéndote realizar consultas y operaciones en la base de datos de manera eficiente.
